Overview

PlayStation(R)Mobile Development Assistant (PSM DevAssistant) is an application for developing a PSM application on PS Vita.

The application must be installed on the applicable device prior to application execution.

Moreover, keys corresponding to the project must be created in advance on the relevant device using the Publishing Utility and a network connection enabled environment is required.

Regarding the setting on each device including installation and details of key creation, refer to the following documents.

  • Setup for Executing with PlayStation(R)Vita
  • Creating a Key to Execute a PSM Application on Actual Devices

PS Vita Version PlayStation(R)Mobile Development Assistant

The following is the screen for the PS Vita version PlayStation(R)Mobile Development Assistant.

Figure 1 PS Vita Version PlayStation(R)Mobile Development Assistant

Select the menu at the bottom right section of Figure 1 to display the option list of the PS Vita version PlayStation(R)Mobile Development Assistant.

Each menu item provides the following features.

Item name Description
Delete Applications Deletes an installed PSM application.
Intellectual Property Notices Displays trademark notices of PlayStation(R)Mobile Development Assistant.
Go to the Developer Forums Starts a web browser and opens the Developer Forums page.
Debug Settings Displays various debug items.

The following items are under [Debug Settings].

Item name Description
Delete Contents Deletes save data (Documents) of all installed PSM applications.
Reset Ticket Information Clears purchase information of all installed PSM applications regarding the In-App Purchase feature.
Show Build Information Displays build information of PlayStation(R)Mobile Development Assistant.

Moreover, operation using buttons of the PS Vita system is possible when Settings application > "System" > "Control with Buttons on PS Vita System" is enabled.

The enter button assignment on PlayStation(R)Mobile Development Assistant will be the same as the enter button assignment on the system software.
